## Navigation Matrix

| Role | Modules Accessible |
|------|-------------------|
| **DEO** | Gate Pass Creation, My GPs |
| **Receiver** | Pending GPs, Receive Confirmation |
| **Security** | Security Confirmation |
| **InventoryController** | Inventory Dashboard, Stock Take, Reconciliation |
| **LossPreventionOfficer** | Exception Reports, Shrinkage Reports, Audit Trail |
| **Operations** | Inventory Dashboard, Reconciliation Reports |
| **Auditor** | All Reports, Audit Trail |
| **Admin** | All Modules + User Management |
